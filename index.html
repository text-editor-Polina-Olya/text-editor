PoLLelya

Интеллектуальная среда для изучения русского языка как иностранного

О проекте

Проект направлен на создание интерактивной платформы по типу Duolingo, адаптированной под специфические сложности русского языка: падежную систему, виды глаголов и подвижное ударение. PoLLelya — это технологическое ядро проекта, обеспечивающее работу с учебными материалами.

Теоретическая база

Разработка опирается на актуальные исследования в области лингводидактики:

Лингвострановедение

Интеграция культурного контекста в каждое упражнение для формирования социокультурной компетенции.

Морфологический анализ

Алгоритмическое выделение окончаний и корней для упрощения понимания структуры слова иностранцами.

Редактор PoLLelya

На текущем этапе реализации основным программным продуктом является редактор PoLLelya. Это инструмент для лингвистов и методистов, позволяющий размечать тексты для будущих упражнений приложения.

Ключевые возможности:

  • Визуальная маркировка грамматических конструкций.
  • Сохранение форматирования при смене режимов работы.
  • Оптимизированный интерфейс для длительной работы над контентом.

Журнал прогресса

Этап 1: Исследование (Март 2026)

Изучение методики РКИ и отбор лексического минимума. Определение структуры будущего приложения.

Этап 2: Прототипирование (Апрель 2026)

Разработка интерфейса PoLLelya. Решение технических проблем с тегированием текста в Python.

Этап 3: Тестирование (Май 2026)

Финальная сборка модуля подготовки контента. Подготовка документации и презентации проекта.
