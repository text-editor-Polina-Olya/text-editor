# Проектная практика

Данный репозиторий содержит материалы по двум учебным заданиям, выполненным в рамках проектной практики. 

## Структура репозитория

* `/index.html` — исходный код информационного портала проекта (HTML/CSS).
* `/editor.py` — программный модуль текстового редактора (Python/Tkinter).
* `отчет базовая часть.docx`
* `отчет вариативная часть.docx`
* `отчет базовая часть.pdf`
* `отчет базовая часть.pdf` — итоговые отчеты по практике.

---

## 1. Командный портал (HTML/CSS)
Информационный веб-ресурс, разработанный для презентации проекта «Мишкова» — системы изучения РКИ для студентов из Латинской Америки.

**Основные разделы сайта:**
* Аннотация и цели проекта.
* Детальное описание концепции.
* Состав команды по отделам с указанием ролей.
* Журнал прогресса разработки.
* Ресурсы и полезные ссылки.

**Технологии:** HTML5, CSS3 (Flexbox, Grid).
**Запуск:** Откройте файл `site/index.html` в любом современном браузере.

---

## 2. Текстовый редактор «PoLLelya» (Python)
Вспомогательный инструмент для контент-менеджеров проекта, позволяющий создавать и редактировать учебные материалы.

**Функционал:**
* Создание, открытие и сохранение текстовых файлов.
* Уведомления о необходимости сохранения при выходе.
* Минималистичный интерфейс для фокусировки на контенте.

**Технологии:** Python 3.x, библиотека Tkinter.
**Запуск:** `python editor/main.py`
