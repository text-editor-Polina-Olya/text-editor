Техническое руководство: Разработка текстового редактора NovaWrite на Python
Введение
Данное руководство описывает процесс создания текстового редактора с использованием библиотеки tkinter. Основное внимание уделено реализации системы динамического форматирования и сохранению состояния интерфейса при смене тем оформления.

1. Архитектура форматирования через систему тегов
В tkinter.Text форматирование реализуется не через свойства объектов, а через теги. Тег — это именованная область текста, к которой привязаны визуальные стили.

Проблема конфликта стилей
Стандартные теги (например, отдельно "Bold" и отдельно "Size 14") конфликтуют при наложении, так как оба пытаются управлять параметром font.

Решение: Реализована система комбинированных тегов. Для каждой уникальной комбинации (шрифт + размер + начертание) создается уникальный идентификатор:
f_{Name}_{Weight}_{Slant}_{Size} (например, f_Arial_bold_italic_14).

2. Механизм перехвата событий (Key Binding)
Стандартный виджет сбрасывает активные стили при перемещении курсора. Для решения этой проблемы был реализован перехват события нажатия клавиш <KeyPress>.

Логика работы:

Программа отслеживает нажатие кнопок форматирования (B, I, U) и сохраняет их состояние в стек current_tags.

При нажатии любой алфавитно-цифровой клавиши стандартная вставка символа блокируется (return "break").

Символ вставляется вручную методом .insert(), и к нему мгновенно применяются все накопленные теги из стека.

3. Сохранение состояния при смене темы (State Management)
При переключении между Dark и Light Mode виджеты tkinter приходится пересобирать для корректного обновления всех системных цветов. Чтобы текст не превращался в "тыкву", используется алгоритм сериализации:

Экспорт данных: Метод .dump() извлекает весь текст вместе с индексами начала и конца всех примененных тегов.

Пересборка: Старое окно редактора удаляется, создается новое с цветами новой темы.

Импорт (Восстановление): Текст вставляется обратно, а специальный цикл заново регистрирует конфигурации тегов (цвета и шрифты) и накладывает их на соответствующие индексы.
